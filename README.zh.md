# Siltflow

面向语言学习的桌面应用——阅读 PDF、查词、用间隔重复（SRS）构建词汇库。

## 设置参考

### FSRS（间隔重复）参数

控制内置 SRS 安排卡片复习的方式。

| 参数 | 默认值 | 说明 |
|---|---|---|
| `request_retention` | `85%` | 卡片到期时你期望回忆起来的目标概率。**越高** → 间隔越短，复习越多，记忆更牢固。**越低** → 间隔越长，复习越少，更容易遗忘。 |
| `maximum_interval` | `365 days` | 两次复习之间的最大天数硬上限。即使算法计算出更长的间隔，也不会超过此值。 |
| `enable_fuzz` | `true` | 给复习间隔增加小幅随机抖动，避免同批卡片在同一天同时到期。建议保持开启。 |
| `enable_short_term` | `true` | 开启后，新卡和遗忘的卡片在进入长期循环之前会经过短时（分钟级）步骤。关闭则直接进入长期排程。 |
| `learning_steps` | `1m, 10m` | **新卡片**的渐进间隔：评 Good → 等 1 分钟 → 再次看到 → 评 Good → 等 10 分钟 → 再次看到 → 毕业进入长期循环。评 Again 会重新开始当前步骤。支持 `m`（分钟）、`h`（小时）、`d`（天）。 |
| `relearning_steps` | `10m` | **遗忘卡片**（毕业后评 Again）的单步抢救间隔。一次短间隔通常就足够把它拉回长期排程。 |

### 键盘快捷键

所有快捷键可在 **设置 → 快捷键** 中自定义。

| 操作 | 默认按键 | 作用域 |
|---|---|---|
| 文档标签页 | `Alt+1` | 全局 |
| 复习标签页 | `Alt+2` | 全局 |
| 大纲标签页 | `Alt+3` | 全局 |
| 笔记标签页 | `Alt+A` | 全局 |
| 摘要标签页 | `Alt+S` | 全局 |
| 切换左面板 | `Alt+[` | 全局 |
| 切换右面板 | `Alt+]` | 全局 |
| 打开设置 | `Ctrl+,` | 全局 |
| 快速添加 | `Ctrl+I` | 全局 |
| 适应宽度 | `Ctrl+E` | PDF 打开时 |
| 开始学习 | `Ctrl+S` | 笔记标签页 |
| 翻卡 / 显示答案 | `Space` | 学习模式 |
| Again（1 分） | `Num1` | 学习模式 |
| Hard（2 分） | `Num2` | 学习模式 |
| Good（3 分） | `Num3` | 学习模式 |
| Easy（4 分） | `Num4` | 学习模式 |
| 播放 / 停止语音 | `Alt+L` | 学习模式 |
| 返回 | `Escape` | 学习模式 |

### TTS（文本转语音）

支持两个引擎：

- **Edge-TTS** — 本地运行，需 `pip install edge-tts`。可为每种语言独立配置语音、语速、音量、音高和二进制路径。
- **MiMo** — 小米云端 API。需要 API Key，支持预设语音模型以及语音设计/克隆。
